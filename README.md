# FilmQuery App

The purpose of this app is to learn how to query database and retrieve results using JDBC.

### Project Description

## Objectives

There are 3 package: 
* app - start application and display menu to users
* database - connects to database and creates sql queries. 
* entities - creates entities/classes which represents database tables

## Topics/Technologies
* Java
* JDBC
* MAVEN
* JUnit 5
* Git command line
* Eclipse IDE
* Object Oriented Design 
  * Polymorphism
  * Inheritance  
* Java Collections Framework
